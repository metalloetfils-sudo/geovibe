# Géovibration 📱

Application de recherche d'établissements de boissons et de bouches par style.

## 📥 Installation sur Android
1. Accédez à la section **Releases** (Sorties) de ce dépôt.
2. Téléchargez le fichier `.apk` de la dernière version.
3. Ouvrez le fichier sur votre smartphone.
4. Si demandé, autorisez l'installation depuis des **sources inconnues** dans les paramètres de votre navigateur ou gestionnaire de fichiers.
